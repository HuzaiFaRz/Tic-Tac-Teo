# 🎮 Tic Tac Toe — React + Vite

A modern and responsive Tic Tac Toe game built with **React** and powered by **Vite** for fast development and smooth performance.

Clean UI. Smart logic. Minimal design.

---

## ⚡ Tech Stack

- React
- Vite
- Modern CSS

---

## ✨ Features

- Turn-based gameplay (X & O)
- Automatic winner detection
- Restart game functionality
- Fully responsive design
- Smooth and minimal UI

---

## 🚀 Getting Started

```bash
npm install
npm run dev
```
